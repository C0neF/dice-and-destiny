## ✨ 游戏简介

**《骰运牌途》** 是一款融合 **卡牌构建** 与 **骰子机制** 的像素风 Roguelite 地牢冒险游戏。

不同于传统的卡牌 Roguelite（如杀戮尖塔），也不同于纯骰子冒险（如骰子地下城），《骰运牌途》将两者深度融合——每张卡牌都拥有**骰子槽位**，骰子的投掷结果会直接放大卡牌的效果。你需要同时经营**卡组**与**骰子收藏**，在每个回合精打细算地分配有限的骰子资源。

一句话概括：**卡牌决定你能做什么，骰子决定你做得多好。**

---

## 🎮 核心玩法

### 🃏 卡牌 × 骰子 = 无限可能

每回合开始时，你的骰子池会自动掷骰。打出卡牌时，它会消耗骰子槽位对应数量的骰子，并将骰子点数乘以卡牌的倍率加到基础效果上。

**举例：**
- 🗡️ **重击**：基础 10 伤害，2 个骰子槽，1.5× 倍率
- 你掷出：[4] [6] [2]
- 打出重击 → 消耗 [4] 和 [6] → 10 + (4+6)×1.5 = **25 伤害！**
- 但只剩 [2] 给剩余卡牌了…

### ⚔️ 连击系统

同一回合打出的每张牌都会获得递增的**连击加成**。第二张 +1，第三张 +2，以此类推。拥有**战鼓遗物**时连击加成翻倍！

### 👁️ 敌人意图

敌人头顶会显示下回合的行动意图：
- ⚔ 攻击（显示伤害值）
- 🛡 防御
- ↑ 增益自身
- ↓ 施加减益
- ⚡ 蓄力大招！

读懂敌人意图，提前布局防守或抢攻——这是生存的关键。

### 🔥 状态效果

战斗中有丰富的状态效果体系：

| 效果 | 说明 |
|------|------|
| 🔥 灼烧 | 每回合受到递减的火焰伤害 |
| ❄️ 冻结 | 跳过下一回合行动 |
| 🟢 中毒 | 每回合受到递减的毒素伤害 |
| 💪 力量 | 每次攻击额外 +N 伤害 |
| 🔻 虚弱 | 造成的伤害降低 25% |
| 💜 易伤 | 受到的伤害增加 50% |
| 💚 再生 | 每回合恢复生命 |
| 🔗 连击 | 当前回合的卡牌加成 |
| 💨 闪避 | 有几率躲避下次攻击 |
| 🛡️ 护盾 | 减少固定伤害 |

## 🏺 遗物系统

在冒险途中收集**遗物**——它们提供持续的被动增益，改变你的战斗方式：

| 遗物 | 稀有度 | 效果 |
|------|--------|------|
| 幸运金币 | ⚪ 普通 | 20% 几率打牌不消耗能量 |
| 血瓶 | ⚪ 普通 | 最大生命 +10，每回合恢复 2 |
| 铁王冠 | 🔵 稀有 | 每回合开始获得 3 护甲 |
| 烈焰之戒 | 🔵 稀有 | 获得火焰骰，攻击额外 +2 |
| 霜冻护符 | 🔵 稀有 | 获得寒冰骰，受到伤害 -1 |
| 毒牙 | 🟣 珍贵 | 获得剧毒骰 |
| 战鼓 | 🟣 珍贵 | 连击加成翻倍 |
| 水晶球 | 🟣 珍贵 | 每回合多抽 1 张牌 |
| 碎镜 | 🟡 Boss | 10% 几率伤害翻倍 |
| 黄金骰 | 🟡 Boss | 骰子最低点数为 3 |

遗物之间可以形成强力组合——比如**战鼓 + 连击卡组**、**黄金骰 + 高倍率卡牌**。

## 🎯 设计理念

《骰运牌途》的核心在于**双重资源管理**：

1. **构筑卡组** — 决定你的战术选择空间
2. **收集骰子** — 决定你的数值上限
3. **回合内分配** — 有限的骰子给哪张牌？这是每回合的核心抉择

这避免了纯卡牌游戏的"完美卡组"问题——再好的卡组也需要好骰子，而骰子操控牌因此具有了独特的战略价值。遗物和连击系统进一步增加了 Build 的多样性，让每次冒险都独一无二。
